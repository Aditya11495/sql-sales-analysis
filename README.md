# sql-sales-analysis
Sales data analysis using joins,functions window,and aggregations.
## Problem Statement
Analyze sales data to identify trends revenue, top customers, and product performance.

## Dataset
Sales transaction dataset containing order, customer, product, and revenue details.

## SQL Concepts Used
- SELECT, WHERE
- GROUP BY, HAVING
- INNER JOIN
- Window Functions (RANK, ROW_NUMBER)
- Aggregations (SUM, AVG)

## Key Analysis Performed
- Monthly sales trend
- Top 5 customers by revenue
- Best-selling products
- Department-wise sales contribution

## Business Insights
- A small group of customers contributes a large portion of revenue
- Certain products show seasonal demand spikes

## Tools Used
- SQL
